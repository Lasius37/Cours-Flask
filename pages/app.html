    Créer l'application

    À la racine du projet, créer un fichier Python nommé app.py. Ce fichier contiendra le code principal de la première application Flask. Le projet devient : 

    mon_projet_flask/
    │
    ├── .venv/
    │
    ├── app.py                  application Flask
    └── requirements.txt

    Dans ce fichier app.py, importer la classe Flask depuis la bibliothèque flask. 

    from flask import Flask

    Créer ensuite une instance de la classe Flask et l'enregistrer dans une variable nommée app. 

    app = Flask(__name__)

    La variable app représente l'application Flask. Elle permettra notamment de définir les différentes pages du site.

    La variable spéciale __name__ indique à Flask dans quel fichier l'application est créée.

    Créer une première page

    Une page Flask est associée à une adresse grâce au décorateur @app.route(). 

    @app.route("/")

def accueil():
    return "Bienvenue sur mon application Flask !"

    La valeur "/" représente la racine du site, c'est-à-dire sa page d'accueil. Lorsque cette adresse est demandée, Flask exécute la fonction accueil().

    Le texte placé après return est ensuite envoyé au navigateur.

    L'application est créée, mais elle n'est pas encore exécutée.

    Erreurs fréquentes

    • Nommer le fichier flask.py : Python risque de confondre ce fichier avec la bibliothèque Flask.
    • Écrire Flask sans majuscule : le nom de la classe doit être respecté exactement.
    • Oublier les parenthèses après accueil : une fonction doit être déclarée avec des parenthèses.
    • Oublier l'indentation après la fonction : l'instruction return doit appartenir au bloc de la fonction.
