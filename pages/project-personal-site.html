Projet : le mini-site personnel

    Dans cette page

    Énoncé

    Vous devez réaliser un mini-site personnel avec Flask.

    • Le site doit présenter une personne réelle ou fictive.
    • Le projet doit utiliser Flask.
    • Le site contiendra au minimum trois pages :
      • une page d'accueil
      • une page de présentation
      • une page de contact ou de centres d'intérêt
    • Chaque page doit être associée à une route et à une vue.
    • Chaque vue doit afficher un template.
    • La navigation doit permettre de passer facilement d'une page à l'autre.
    • Les pages doivent partager une présentation cohérente basée sur un template commun :
      • un en-tête commun
      • un menu de navigation
      • un pied de page
      • une feuille de style commune
    • Le contenu doit comporter :
      • un titre clair sur chaque page
      • des textes structurés
      • au moins une image
      • des liens de navigation visibles
    • Le projet doit fonctionner avec le serveur de développement Flask.

    Conseils de réalisation

    • Créer un nouveau projet Python avec un environnement virtuel.
    • Installer Flask et enregistrer les dépendances dans requirements.txt.
    • Créer le fichier app.py.
    • Créer les routes et les vues nécessaires :
      • / : page d'accueil
      • /about : page de présentation
      • /contact , /hobbies ou une autre page complémentaire
    • Créer les fichiers HTML nécessaires dans le dossier templates.
    • Créer un fichier base.html contenant :
      • la structure générale de la page
      • le menu de navigation
      • le pied de page
      • un bloc destiné au contenu propre à chaque page
    • Faire hériter les autres templates de base.html.
    • Créer un dossier static contenant :
      • une feuille de style CSS
      • au moins une image
      • éventuellement un fichier JavaScript
    • Soigner le contenu :
      • présenter la personne choisie
      • indiquer son parcours, ses goûts, ses compétences ou ses projets
      • utiliser des titres, des paragraphes et éventuellement des listes
      • ajouter une image pertinente
    • Personnaliser l'apparence du site :
      • choisir une palette de couleurs cohérente
      • utiliser une police lisible
      • prévoir des espacements réguliers
      • rendre le menu clairement identifiable
    • Tester le projet avec le serveur Flask et vérifier que :
      • toutes les pages s'affichent
      • tous les liens fonctionnent
      • la feuille de style est chargée
      • les images apparaissent correctement
      • aucune erreur de type erreur 404 ou TemplateNotFound n'apparaît
