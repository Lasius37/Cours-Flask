    Créer le projet

    Dans PyCharm, créer un nouveau projet Python et choisir son emplacement.

    Le nom du dossier doit décrire clairement le projet. Il est préférable d'éviter les espaces, les accents et les caractères spéciaux. Par exemple : 

    mon_projet_flask

    PyCharm ouvre ensuite ce dossier comme racine du projet. Tous les fichiers de l'application seront progressivement ajoutés dans ce dossier.

    Lors de la création du projet, PyCharm peut créer automatiquement un environnement virtuel nommé .venv. Il faut vérifier que cette option est activée et que le bon interpréteur Python est sélectionné.

    Installer Flask

    Une fois l'environnement virtuel sélectionné et activé, Flask peut être installé avec pip. 

    pip install Flask

    La commande installe Flask ainsi que les bibliothèques nécessaires à son fonctionnement. Elles sont uniquement ajoutées à l'environnement virtuel du projet. Il n'est donc pas nécessaire d'installer Flask séparément sur l'ensemble de l'ordinateur.

    Pour vérifier les bibliothèques installées (dont Flask) : 

    pip list

    Enregistrer les dépendances

    Les bibliothèques utilisées par le projet peuvent être enregistrées dans un fichier requirements.txt, qui contient le nom et la version des bibliothèques installées. 

    pip freeze > requirements.txt

    Il permet de recréer plus tard le même environnement, notamment après avoir récupéré le projet sur un autre ordinateur : 

    pip install -r requirements.txt

    Observer la structure du projet

    À ce stade, le projet ne contient pas encore d'application Flask. Sa structure reste volontairement minimale. 

    mon_projet_flask/
    │
    ├── .venv/                  environnement virtuel du projet
    │
    └── requirements.txt        bibliothèques nécessaires au projet

    Erreurs fréquentes

    • Ne pas installer d'environnement virtuel : risque de conflit entre les bibliothèques importées.
    • Oublier de mettre à jour requirements.txt après l'installation d'une nouvelle bibliothèque : le projet ne pourra pas être recréé complètement.
